Page67.1 用本节方法判断下列同余式是否有解：
(a) $x^2 \equiv 429 \pmod{563};$
(b) $x^2 \equiv 680 \pmod{769};$
(c) $x^2 \equiv 503 \pmod{1013}$
(其中 (503, 563, 769, 1013) 都是素数)

(a) 由于 (429 = 3 \times 11 \times 13)，
(\left(\dfrac{429}{563}\right)=\left(\dfrac{3 \times 11 \times 13}{563}\right)=\left(\dfrac{3}{563}\right)\left(\dfrac{11}{563}\right)\left(\dfrac{13}{563}\right))

(\left(\dfrac{3}{563}\right) = -\left(\dfrac{563}{3}\right) = -\left(\dfrac{2}{3}\right) = -(-1)=1)，

(\left(\dfrac{11}{563}\right) = -\left(\dfrac{563}{11}\right) = -\left(\dfrac{2}{11}\right) = -(-1)=1)，

(\left(\dfrac{13}{563}\right) = \left(\dfrac{563}{13}\right) = \left(\dfrac{4}{13}\right)=\left(\dfrac{2}{13}\right)^2=1)，
故 (\left(\dfrac{429}{563}\right)=1)，同余式有解。

(b) (680 = 2^3 \times 5 \times 17)。
由于 (769 \equiv 1 \pmod{8})，(\left(\dfrac{2}{769}\right)=1)。又由二次互反律

(\left(\dfrac{5}{769}\right)=\left(\dfrac{769}{5}\right)=\left(\dfrac{4}{5}\right)=\left(\dfrac{2}{5}\right)^2=1)，
(\left(\dfrac{17}{769}\right)=\left(\dfrac{769}{17}\right)=\left(\dfrac{2}{17}\right)^2=1)，

故 (\left(\dfrac{680}{769}\right)=\left(\dfrac{2}{769}\right)^3\left(\dfrac{5}{769}\right)\left(\dfrac{8}{769}\right)=1)，同余式有解。

(c) (x^2 \equiv 503 \pmod{1013})
(\left(\dfrac{503}{1013}\right)=\left(\dfrac{1013}{503}\right)=\left(\dfrac{7}{503}\right))，

(\left(\dfrac{7}{503}\right)=-\left(\dfrac{503}{7}\right)=-\left(\dfrac{6}{7}\right)=-\left(\dfrac{2}{7}\right)\left(\dfrac{3}{7}\right))，
(\left(\dfrac{2}{7}\right)=1)，(\left(\dfrac{3}{7}\right)=-1)，

故 (\left(\dfrac{7}{503}\right)=1)，同余式有解。

Page67.2 求出以 (-2) 为平方剩余的素数的一般表达式；以 (-2) 为平方非剩余时的素数的一般表达式。

(\left(\dfrac{-2}{p}\right) = \left(\dfrac{-1}{p}\right)\left(\dfrac{2}{p}\right))。
由 (\left(\dfrac{-1}{p}\right)=1) 当 (p\equiv1\pmod4)，(\left(\dfrac{-1}{p}\right)=-1) 当 (p\equiv3\pmod4)；
(\left(\dfrac{2}{p}\right)=1) 当 (p\equiv\pm1\pmod8)，(=-1) 当 (p\equiv\pm3\pmod8)。
组合得：
(\left(\dfrac{-2}{p}\right)=1 \iff p\equiv1,3\pmod8)；
(\left(\dfrac{-2}{p}\right)=-1 \iff p\equiv5,7\pmod8)。
故以 (-2) 为平方剩余的素数 (p) 满足 (p\equiv1) 或 (3\pmod8)；
以 (-2) 为平方非剩余的素数 (p) 满足 (p\equiv5) 或 (7\pmod8)。

Page67.3 设 ( n ) 是正整数，( 4n + 3 ) 及 ( 8n + 7 ) 都是素数，说明：
[
2^{4n+3} \equiv 1 \pmod{8n + 7}.
]
由此证明
[
23 \mid (2^{11} - 1), \quad 47 \mid (2^{23} - 1), \quad 503 \mid (2^{251} - 1).
]

令 ( p = 4n+3 )，( q = 8n+7 )，则 ( q = 2p+1 )。由于 ( q ) 是素数，且
( q \equiv 7 \pmod{8} )，故由二次互反律，Legendre 符号
[
\left(\frac{2}{q}\right) = 1 \quad (q \equiv \pm1 \pmod{8}).
]
从而
[
2^{(q-1)/2} = 2^p \equiv \left(\frac{2}{q}\right) \equiv 1 \pmod{q}.
]
即 ( 2^{4n+3} \equiv 1 \pmod{8n+7} )，命题得证。

下面利用该结论证明题中三个整除关系：

取 ( n = 2 )，则 ( 4n+3 = 11 )，( 8n+7 = 23 )，均为素数。于是
( 2^{11} \equiv 1 \pmod{23} )，即 ( 23 \mid (2^{11}-1) )。

取 ( n = 5 )，则 ( 4n+3 = 23 )，( 8n+7 = 47 )，均为素数。于是
( 2^{23} \equiv 1 \pmod{47} )，即 ( 47 \mid (2^{23}-1) )。

取 ( n = 62 )，则 ( 4n+3 = 251 )，( 8n+7 = 503 )，均为素数。于是
( 2^{251} \equiv 1 \pmod{503} )，即 ( 503 \mid (2^{251}-1) )。

Page70.1 求以 (\pm 3) 为平方剩余的素数的一般表达式；什么素数以 (\pm3) 为平方非剩余。

(p=2) 时，(3\equiv1)，(-3\equiv1)，故 (\pm3) 均为平方剩余。下面
对于奇素数 (p \neq 3)，
[
\left(\frac{-3}{p}\right) = \left(\frac{-1}{p}\right)\left(\frac{3}{p}\right), \quad
\left(\frac{-1}{p}\right) = (-1)^{\frac{p-1}{2}} = \begin{cases}
1, & p \equiv 1 \pmod{4} \
-1, & p \equiv 3 \pmod{4}
\end{cases}
]
二次互反律给出：
[
\left(\frac{3}{p}\right) = \left(\frac{p}{3}\right) (-1)^{\frac{p-1}{2}}, \quad \left(\frac{p}{3}\right) = \begin{cases}
1, & p \equiv 1 \pmod{3} \
-1, & p \equiv 2 \pmod{3}
\end{cases}
]

case 1 (\pm 3) 均为平方剩余，即
[
\left(\frac{3}{p}\right)=1,\quad \left(\frac{-3}{p}\right)=1.
]
由 (\left(\dfrac{-3}{p}\right)=\left(\dfrac{-1}{p}\right)\cdot1=1) 得 (\left(\dfrac{-1}{p}\right)=1)，故 (p \equiv 1 \pmod{4})。此时 ((-1)^{\frac{p-1}{2}}=1)，所以
[
\left(\frac{3}{p}\right)=\left(\frac{p}{3}\right)=1 \implies p \equiv 1 \pmod{3}.
]
联立 (p \equiv 1 \pmod{4}) 与 (p \equiv 1 \pmod{3})，由中国剩余定理得 (p \equiv 1 \pmod{12})。

case 2 (\pm 3) 均为平方非剩余，即
[
\left(\frac{3}{p}\right)=-1,\quad \left(\frac{-3}{p}\right)=-1.
]
代入 (\left(\dfrac{-3}{p}\right)=\left(\dfrac{-1}{p}\right)(-1)=-1) 得 (\left(\frac{-1}{p}\right)=1)，仍需 (p \equiv 1 \pmod{4})。此时 ((-1)^{\frac{p-1}{2}}=1)，于是
[
\left(\frac{3}{p}\right)=\left(\frac{p}{3}\right)=-1 \implies p \equiv 2 \pmod{3}.
]
联立 (p \equiv 1 \pmod{4}) 与 (p \equiv 2 \pmod{3})，解得 (p \equiv 5 \pmod{12})。

Page64.1 在 (4) 式中，若 ( p^\alpha \mid A )，试求出 (4) 的一切解来。

(4) 现在可写成 (y^2\equiv 0\pmod{p^\alpha})。
从而 (y=kp^t)，对于一些 (k,t\in\mathbb{Z})。即 (y^2=k^2p^{2t})。要求 ( p^\alpha \mid y^2 )，即 ( 2t \ge \alpha )，亦即 ( t \ge \lceil \alpha/2 \rceil )。因此 ( y ) 必须是 ( p^{\lceil \alpha/2 \rceil} ) 的倍数。模 ( p^\alpha ) 意义下，所有解为
[
y \equiv 0,; p^{\lceil \alpha/2 \rceil},; 2p^{\lceil \alpha/2 \rceil},; \dots,; (p^{\alpha - \lceil \alpha/2 \rceil} - 1) p^{\lceil \alpha/2 \rceil} \pmod{p^\alpha}.
]
即解共有 ( p^{\alpha - \lceil \alpha/2 \rceil} ) 个，可表示为
[
y = k \cdot p^{\lceil \alpha/2 \rceil}, \quad k = 0,1,\dots, p^{\alpha - \lceil \alpha/2 \rceil} - 1.
]

Page64.2 证明：同余式
[
ax^2 + bx + c \equiv 0 \pmod{m}, \qquad (2a, m) = 1
]
有解的充分必要条件是
[
x^2 \equiv q \pmod{m}, \qquad q = b^2 - 4ac
]
有解，并且前一同余式的一切解可由后一同余式的解导出。

(必要性) 设 ( x_0 ) 满足 ( ax_0^2 + bx_0 + c \equiv 0 \pmod{m} )。将同余式两边乘以 ( 4a )，加上 ( b^2 ) 得
[
4a^2 x_0^2 + 4ab x_0 + b^2 \equiv b^2 - 4ac \pmod{m},
]
即
[
(2a x_0 + b)^2 \equiv q \pmod{m}.
]
因此 ( y_0 = 2a x_0 + b ) 是 ( y^2 \equiv q \pmod{m} ) 的一个解。

(充分性) 设 ( y_0 ) 满足 ( y_0^2 \equiv q \pmod{m} )。由于 ( (2a, m) = 1 )，存在整数 ( \bar{a} ) 使得 ( 2a \bar{a} \equiv 1 \pmod{m} )。令
[
x_0 \equiv \bar{a} (y_0 - b) \pmod{m}.
]
则 ( 2a x_0 + b \equiv y_0 \pmod{m} )。于是
[
(2a x_0 + b)^2 \equiv y_0^2 \equiv q = b^2 - 4ac \pmod{m}.
]
展开左边：
[
4a^2 x_0^2 + 4ab x_0 + b^2 \equiv b^2 - 4ac \pmod{m},
]
消去 ( b^2 ) 得
[
4a^2 x_0^2 + 4ab x_0 \equiv -4ac \pmod{m},
]
即
[
4a(ax_0^2 + bx_0 + c) \equiv 0 \pmod{m}.
]
由 ( (2a, m) = 1 ) 知 ( m ) 为奇数且 ( (a, m) = 1 )，从而 ( (4a, m) = 1 )。因此可两边乘以 ( 4a ) 的逆元，得到
[
ax_0^2 + bx_0 + c \equiv 0 \pmod{m}.
]
故 ( x_0 ) 是原同余式的解。

若 ( y ) 遍历 ( y^2 \equiv q \pmod{m} ) 的全部解，则 ( x \equiv (2a)^{-1}(y - b) \pmod{m} ) 遍历原同余式的全部解（这里 ( (2a)^{-1} ) 表示模 ( m ) 下 ( 2a ) 的逆元）。因此原同余式的一切解可由后一同余式的解导出。

Page65.1 求出 mod 37 的平方剩余和非平方剩余。

37 是素数，平方剩余共有 (\frac{37-1}{2}=18) 个。计算 (1^2,2^2,\dots,18^2) 模 37：

[
\begin{aligned}
1^2&\equiv1,& 2^2&\equiv4,& 3^2&\equiv9,& 4^2&\equiv16,&
5^2&\equiv25,& 6^2&\equiv36,\7^2&\equiv12,& 8^2&\equiv27,&
9^2&\equiv7,& 10^2&\equiv26,& 11^2&\equiv10,& 12^2&\equiv33,\
13^2&\equiv21,& 14^2&\equiv11,& 15^2&\equiv3,& 16^2&\equiv34,&17^2&\equiv30,& 18^2&\equiv28.
\end{aligned}
]
故平方剩余为
[
{1,3,4,7,9,10,11,12,16,21,25,26,27,28,30,33,34,36}.
]
平方非剩余为
[
{2,5,6,8,13,14,15,17,18,19,20,22,23,24,29,31,32,35}.
]

Page65.2

(i) 模 (p) 的既约剩余系中一定有平方剩余及平方非剩余存在。

(ii)（乘性）证明零个平方剩余的乘积是平方剩余，平方剩余与平方非剩余的乘积是平方非剩余。

(iii) 平方剩余与平方非剩余的个数各为 (\frac{p-1}{2})。

证明：
(i) 模 (p) 的既约剩余系构成乘法群 ((\mathbb{Z}/p\mathbb{Z})^\times)，阶为 (p-1\ge 2)（因为 (p) 为奇素数）。该群是循环群，设 (g) 为一个原根。则平方剩余恰为 (g) 的偶次幂，平方非剩余为 (g) 的奇次幂。由于 (p-1\ge 2)，至少存在一个偶次幂和一个奇次幂，故两者都存在。
（实际上 (\left(\frac{1}{p}\right)=1)，且存在 (a) 使 (\left(\frac{a}{p}\right)=-1)，因为 (\sum_{a=1}^{p-1}\left(\frac{a}{p}\right)=0) 且 (\left(\frac{a}{p}\right)) 非全为 (1)。）

(ii) 设 (a,b) 为平方剩余，则存在 (x,y) 使 (x^2\equiv a,;y^2\equiv b\pmod{p})，于是 ((xy)^2\equiv ab)，故 (ab) 也是平方剩余；设 (a) 为平方剩余，(b) 为平方非剩余。若 (ab) 是平方剩余，则存在 (z^2\equiv ab)。由 (a) 可逆，得 (b\equiv (z\cdot a^{-1})^2)，矛盾。故 (ab) 是平方非剩余。
（也即 (\left(\frac{ab}{p}\right)=\left(\frac{a}{p}\right)\left(\frac{b}{p}\right))。）

(iii) 由 (i) 知至少各有一个。考虑映射 (\varphi: (\mathbb{Z}/p\mathbb{Z})^\times \to (\mathbb{Z}/p\mathbb{Z})^\times,; \varphi(x)=x^2)。核为 ({1,-1})，阶为 (2)，故像的阶为 (\frac{p-1}{2})。像即为所有平方剩余，所以平方剩余恰有 (\frac{p-1}{2}) 个。从而平方非剩余也有 ((p-1)-\frac{p-1}{2}=\frac{p-1}{2}) 个。

Page65.4 证明 (x^2+1\equiv0\pmod{p})（(p=4m+1)）的解为 (x\equiv\pm(2m)!\pmod{p})。

由 Wilson 定理，((p-1)!\equiv-1\pmod{p})。而
[
p-1=4m,\quad (p-1)! = 1\cdot2\cdots(2m)\cdot(2m+1)\cdots(4m).
]
对于 (k=1,2,\dots,2m)，有 (p-k\equiv -k\pmod{p})，且当 (k) 取遍 (1,\dots,2m) 时，(p-k) 取遍 (2m+1,\dots,4m)。因此
[
(2m+1)(2m+2)\cdots(4m) \equiv (-1)(-2)\cdots(-2m) = (-1)^{2m}(2m)! \pmod{p}.
]
由于 (2m) 为偶数，((-1)^{2m}=1)，故
[
(p-1)! \equiv (2m)! \cdot (2m)! = \bigl((2m)!\bigr)^2 \pmod{p}.
]
于是 (\bigl((2m)!\bigr)^2 \equiv -1\pmod{p})，即 (x\equiv\pm(2m)!) 满足 (x^2\equiv-1)。因此原同余式的解为
[
x \equiv \pm 1\cdot2\cdot\cdots\cdot(2m) \pmod{p}.
]
